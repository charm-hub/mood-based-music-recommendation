This project recommends songs based on the user's selected mood using Spotify audio features.

Features
- Mood classification using audio features
- Interactive UI using ipywidgets
- Dropdown-based mood selection
- Real-time song recommendations

# Technologies Used
- Python
- Pandas, NumPy
- Scikit-learn
- ipywidgets
- Google Colab

# Dataset
- Spotify Tracks Dataset (`tracks.csv`)

## How to Run
1. Open the notebook in Google Colab
2. Upload `tracks.csv`
3. Run all cells
4. Select a mood from dropdown to get recommendations

# Moods Supported
- Happy
- Sad
- Energetic
